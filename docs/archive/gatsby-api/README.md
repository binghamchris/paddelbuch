# Archived Gatsby API Generators

This directory contains the archived Gatsby JSON page generators that were used in the original Gatsby-based implementation of Paddel Buch.

These files have been archived for reference purposes during the migration to Jekyll. The functionality has been replaced by:

- Jekyll plugins in `_plugins/api_generator.rb` for JSON API generation
- Jekyll plugins in `_plugins/tile_generator.rb` for spatial tile generation

## Original Purpose

These JavaScript files defined GraphQL queries and data transformations for generating JSON API endpoints using the `@binghamchris/gatsby-plugin-json-pages` Gatsby plugin.

## Migration Date

Archived on: December 2025

## Note

These files are no longer used in the Jekyll-based build process and are kept only for historical reference.
